# Helios Event Website – Brand Guidelines

## Overview

This document defines the official branding elements for the Helios Event website including primary colors, CTA colors, typography, and usage guidelines. Visual color swatches are included next to each color code for quick identification.

---

# 1. Primary Brand Color

Primary Color

<span style="display:inline-block;width:18px;height:18px;background:#ADC905;border-radius:4px;border:1px solid #ccc;margin-right:8px;"></span> **#ADC905**

Usage:

- Brand identity
- Logo highlights
- Icons
- Section accents
- UI highlights

# 5. Typography

## Heading Font

Font: **Poppins**

Usage:

- Hero titles
- Section headings
- Service titles
- Portfolio titles

Recommended Weights

- 600
- 700
- 800

---

## Body Font

Font: **Inter**

Usage:

- Paragraph text
- Blog content
- Navigation
- Forms

Recommended Weights

- 400
- 500
- 600

## Design Philosophy

The Helios Event website should deliver a world-class UI/UX experience.

The design must feel:

- premium
- modern
- cinematic
- visually engaging
- image-driven

The website should showcase events through strong visuals, immersive layouts, and modern typography.

User experience should focus on:

- fast navigation
- clear service presentation
- strong portfolio showcase
- easy lead generation
- mobile-first responsiveness

The goal is to create a website that reflects the scale and professionalism of Helios Event Productions.
