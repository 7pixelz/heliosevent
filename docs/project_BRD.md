## 1. Project Overview

Helios Event is a professional event management company delivering a wide range of services including corporate events, exhibitions, conferences, government protocol events, sports events, weddings, and digital marketing activations. The existing website ([https://www.heliosevent.in](https://www.heliosevent.in)) requires a complete rebuild to better represent the brand, improve user experience, and strengthen its digital presence.

The new website will act as a primary digital platform to showcase Helios Event’s services, event portfolio, and industry expertise while generating qualified inquiries from corporate clients, organizations, and individuals planning events.

The redesigned platform will focus on modern design, strong visual storytelling, improved SEO architecture, and efficient lead generation.

---

## 2. Project Goals

- Rebuild the existing website with a modern, premium, and visually engaging UI/UX.
- Position Helios Event as a trusted **corporate and large‑scale event management company in Chennai**.
- Showcase the company’s event portfolio with structured categories and event galleries.
- Increase qualified inquiries for corporate events, exhibitions, conferences, and weddings.
- Improve SEO rankings for event management keywords in Chennai and surrounding industrial areas.
- Support location‑based SEO through scalable city and area landing pages.
- Provide dedicated sections for **career opportunities and vendor partnerships**.
- Implement a blog system to support content marketing and organic traffic growth.
- Ensure the website is **mobile‑first, fast loading, and optimized for performance**.
- Provide an easy‑to‑use admin panel for managing services, portfolio, blogs, cities, and locations.

## 9. URL Structure and Redirect Requirements

The current website uses service URLs with a `/services/` prefix.

Example:

Current URL

[https://www.heliosevent.in/services/exhibition-organizer-in-chennai/](https://www.heliosevent.in/services/exhibition-organizer-in-chennai/)

New URL Structure

[https://www.heliosevent.in/exhibition-organizer-in-chennai/](https://www.heliosevent.in/exhibition-organizer-in-chennai/)

The `/services/` prefix will be removed to maintain a clean and SEO‑friendly structure.

All existing URLs must be redirected using **301 permanent redirects** to preserve SEO value, backlinks, and search engine rankings.

- Enable or disable location pages

This allows the website to scale easily for location‑based SEO.

## 20. Project Scope

The following items are included in this project:
